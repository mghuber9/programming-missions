Mission 1: Talk to the Computer

Difficulty: ⭐☆☆☆☆

Goal: Make the computer say something.

Mission Brief

Computers do not speak unless we give them instructions. In this mission, you will learn how to make JavaScript print a message.

Observe

This code tells the computer to say something:

console.log("Hello, computer!");

Predict

Before running the code, what do you think the computer will say?

Experiment

Change the words inside the quotation marks. Then click Run.


Output

Challenge

Make the computer say:

I am learning JavaScript!

Mission Debrief

console.log() is a JavaScript command that prints a message. The message goes inside quotation marks.

Bonus Exploration

Try making the computer print three different messages.
