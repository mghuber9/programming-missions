Mission 1: Messages from the Computer

Difficulty: ⭐☆☆☆☆

Goal: Use JavaScript to display messages from your program.

Mission Brief

Imagine you built a robot and gave it instructions.

But the robot never tells you what it's doing.

  • Did it understand?
  • Is it working?
  • Did it make a mistake?

Programmers have the same problem.

We need a way for the computer to send messages back to us.

Three Places JavaScript Can Send Messages

JavaScript Tool Sends a message to... Used for...
console.log() The programmer Testing and debugging
alert() The user Popup messages
document.write() The webpage Writing directly on the page

In this mission, we will focus on console.log().

Why This Exists

When programmers write code, they constantly want to know what their program is doing.

console.log() lets a program send messages to the programmer.

Normally those messages appear in the browser's Developer Console.

Since you're just learning, our Mission Workspace shows those console messages below so they're easier to see.

Observe

This program sends one message to the programmer console:
