Mission 1: Messages from the Computer

Difficulty: ⭐☆☆☆☆

Goal: Use JavaScript to display messages from your program.

Mission Brief

Imagine you built a robot and gave it instructions.

But the robot never tells you what it's doing.

  • Did it understand?
  • Is it working?
  • Did it make a mistake?

Programmers have the same problem.

We need a way for the computer to send messages back to us.

One of the easiest ways to do that in JavaScript is with console.log().

Why This Exists

When programmers write code, they constantly want to know what their program is doing.

JavaScript provides a tool called console.log() that displays messages.

Normally those messages appear in the browser's Developer Console.

Since you're just learning, our Mission Workspace displays those messages below so they're easier to see.

Observe

This program sends one message. 

console.log("Hello, computer!");

Click Run Code.

Notice that the message appears in the Mission Workspace below.

Predict

Before clicking Run Code, what message do you think this program will display? 

console.log("Pizza");

Try to predict the answer before running it.

Mission Debrief

Today you learned your first programming tool for communicating with your program.

console.log() lets your program send messages back to you.

Professional programmers use this tool every day to understand what their programs are doing.
